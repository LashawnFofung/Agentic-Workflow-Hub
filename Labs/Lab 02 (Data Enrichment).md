# 🧪 Lab 02 (Data Enrichment)

## Data Structuring

**Lead Enrichment Agent**
Takes a LinkedIn URL and uses Gemini to find the company's tech stack and latest news.

## n8n Prompt

Build an agentic workflow that takes a URL via a webhook, uses a Scraping tool to read the page, and uses Gemini to extract the company name, industry, and a 1-sentence summary of what they do, then saves it to a Google Sheet.
