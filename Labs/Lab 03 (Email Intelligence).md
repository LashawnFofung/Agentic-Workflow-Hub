# 🧪 Lab 03 (Email Intelligence)

## Human-in-the-loop

**Email Triage Agent**
Drafts replies to emails but waits for you to click a "Approve" button in Slack before sending.



## n8n Prompt

Create an agent that checks my Gmail for new messages. Use an LLM to categorize the email as 'Urgent', 'Inquiry', or 'Spam'. If it's an 'Inquiry', have the agent draft a reply based on a provided FAQ text and save it as a draft.
